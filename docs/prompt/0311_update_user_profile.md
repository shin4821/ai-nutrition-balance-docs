1. 질환, 약물 탭
   1.1. 질환과 알레르기 작성 시 placeholder에 '쉼표로 구분'하라고 되어있는데 뛰어쓰기와 쉼표 작성이 안돼.
   1.2. '사진으로 간편 등록' 은 복용 양물과 영양제의 추가 버튼을 눌렀을 때 나오는 칸으로 이동시켜줘.
   1.3. 그리고 복용 양물과 영양제의 '복용 시간'은 '식습관,목표 탭의 생활패턴의 취침 시간, 기상 시간에 사용된 시간 UI를 적용해줘.
