'C:\Workspace\nutrition-balance-ai\app' 하위에 analyzer_water.py 파일 만들어서 아래 방식대로 LLM 피드백을 줄 수 있도록 해줘.

// 피드백 방식
"물 섭취량을 분석하고 개인 맞춤형 피드백을 제공하는 app/analyzer_water.py를 작성해줘.

    필수 포함 로직:

        추출 정보: LLM을 이용해 섭취한 물의 양(ml)을 숫자로 추출해줘.

        개인화된 피드백 (핵심): user_profile.json에 저장된 사용자의 지병 정보를 참고해서 피드백을 생성해야 해.

            예시 (고혈압): '오늘 짠 음식을 드셨으니 물을 두 컵 더 드시면 혈압 관리에 도움이 돼요!'

            예시 (신장질환): '수분 섭취를 조절해야 하는 질환이 있으니 너무 과하지 않게 적정량만 드세요.'

        반환 형식: Pydantic 모델을 사용하여 amount_ml, feedback을 반환하도록 짜줘.

        참고: 기존 analyzer.py와 비슷한 구조로 async/await를 사용해줘."


//지병 연계 피드백 로직 설계 (Tip)
단순히 물 양만 계산하는 게 아니라, DB에 저장된 오늘의 식사 기록이나 지병 정보를 프롬프트에 같이 넣어줘야 합니다.
